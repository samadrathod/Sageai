Project: SAGE вЂ“ Smart Adaptive Guidance Engine
Vision
Build a personal AI assistant called SAGE (Smart Adaptive Guidance Engine), inspired by the functionality and elegance of Jarvis but designed as an original, modular, and extensible AI system. SAGE should act as a voice-enabled, memory-based personal assistant capable of conversation, task automation, information retrieval, and future expansion into a full desktop AI ecosystem.
Primary Goals

Natural voice conversations
Persistent memory and personalization
Tool and task execution
Modern futuristic user interface
Modular architecture for easy expansion
Production-quality, clean, maintainable code

Development Rules

Build incrementally.
Do NOT generate the entire project at once.
Implement one feature at a time.
After each step, explain:

Why the file exists
What the code does
How it works internally


Ask me to test each step before proceeding.
Follow industry-standard coding practices.
Write clean, human-readable, well-commented code.
Use modular, reusable components.
Avoid hardcoding values.
Prepare the architecture for future desktop automation and AI agent capabilities.


Workflow
For every step:

Explain the objective.
Generate only the files required for that step.
Explain every line of important code.
Provide setup instructions.
Wait for my confirmation and test results before moving to the next phase.

Begin with Phase 1 only:

Create the folder structure.
Set up FastAPI.
Create a basic futuristic chat interface.
Connect a simple chat endpoint.
Wait for my approval before adding any other features.
